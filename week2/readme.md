## 📖 도움말 및 매뉴얼 확인 명령어
명령어의 사용법이나 옵션이 기억나지 않을 때 사용하는 명령어들입니다.

* `man [명령어]`: **Manual**의 약자로, 명령어의 상세한 공식 매뉴얼 페이지를 엽니다. (종료: `q`)
* `info [명령어]`: `man` 페이지보다 좀 더 구조화되어 있고 상세한 하이퍼링크 형태의 정보를 제공합니다.
* `[명령어] --help`: 명령어의 핵심 사용법과 자주 쓰는 옵션들만 터미널 화면에 간단히 요약해서 출력합니다.

## ⚙️ 시스템 관리 및 패키지 설치 명령어
명령어 앞의 `#` 기호는 **Root(최고 관리자)** 권한으로 실행해야 함을 의미합니다.

* `# passwd root`: 시스템 최고 관리자인 `root` 계정의 비밀번호를 설정하거나 변경합니다.
* `# apt-get update`: 설치 가능한 소프트웨어 패키지들의 최신 목록을 서버로부터 가져와 업데이트합니다. (패키지 설치 전 필수 실행)
* `# apt install gcc`: C/C++ 프로그램 컴파일러인 `gcc` 패키지를 다운로드하고 설치합니다.

## 📂 디렉토리 이동 및 경로 기호
명령어 앞의 `$` 기호는 **일반 사용자** 권한으로 실행 중임을 의미합니다.

* `$ cd ~`: 현재 로그인한 사용자의 **홈 디렉토리**로 이동합니다. (`cd`는 Change Directory의 약자)

**[경로를 나타내는 특수 기호]**
* `.`: **현재 디렉토리** (예: `./script.sh` -> 현재 폴더의 스크립트 실행)
* `..`: **부모 디렉토리** (현재 위치에서 한 단계 상위 폴더)
* `~`: **홈 디렉토리** (사용자의 기본 개인 작업 공간)
* `/`: **루트 디렉토리** (리눅스 파일 시스템의 최상위 경로)

## 🔍 파일 및 디렉토리 목록 확인 (ls 명령어)
현재 위치한 디렉토리 안의 내용물을 확인합니다.

* `ls`: 파일과 디렉토리의 이름만 간단히 목록으로 보여줍니다.
* `ls -l`: 파일의 권한, 소유자, 파일 크기, 수정 날짜 등 **상세 정보(long format)**를 함께 보여줍니다.
* `ls -al`: `ls -l`의 결과에 더해, 이름이 `.`으로 시작하는 **숨김 파일(all)**까지 모두 보여줍니다.
* `ll`: 대부분의 리눅스 배포판에서 `ls -al` 또는 `ls -l`을 편하게 쓰기 위해 기본으로 설정해 둔 **단축 명령어**입니다.
* `alias`: 현재 시스템에 설정된 단축 명령어(별칭) 목록을 확인하거나, 새로운 단축 명령어를 직접 만들 때 사용합니다. (예: `alias ll='ls -al'`)
